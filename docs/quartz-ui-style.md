# Quartz UI Style Note

Короткая памятка по визуальному языку публичного Quartz-сайта Project Andromeda.

## Базовая форма

- Основной язык интерфейса: `rounded rectangles`.
- Мелкие кнопки, toggles, бейджи, табличные акценты и интерактивные чипы делаются как скруглённые прямоугольники.
- Не использовать pill-капсулы и круглые кнопки как базовый мотив, если нет особой причины.

## Применение

- Бейджи под названиями сущностей должны быть компактными, прямоугольными и легко сканироваться в строке таблицы.
- Переключатели раскрытия должны иметь стабильную геометрию в обоих состояниях.
- При смене состояния у toggle меняются только направление стрелки и подсветка, без скачков размера или положения.
- Контролы фильтрации и сортировки должны использовать те же радиусы, что и бейджи/тогглы, если нет явной причины для отклонения.

## Таблицы и каталоги

- Summary-строки таблиц выравнивать по вертикальному центру, если строка состоит из нескольких коротких смысловых колонок.
- В name-колонках допустим второй уровень информации под заголовком строки: теги, частота, школа, навык и другие компактные признаки.
- Полное описание и вторичный длинный контент выносить в раскрывающийся detail-block, а не пытаться держать весь текст в одной строке.
